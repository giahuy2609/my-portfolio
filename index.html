Personal Portfolio Website (Hồ sơ năng lực trực tuyến)

06/2026 - Hiện tại

Mục đích dự án: Xây dựng trang web hồ sơ năng lực cá nhân nhằm ứng dụng kiến thức phát triển giao diện Front-end, đồng thời số hóa CV để tạo ấn tượng trực quan và chuyên nghiệp với nhà tuyển dụng.

Tính năng chính: Hiển thị thông tin cá nhân, mục tiêu nghề nghiệp và các chứng chỉ đạt được; tích hợp thanh menu điều hướng thông minh cuộn mượt mà; giao diện thiết kế theo phong cách tối (Dark Mode) tối ưu trải nghiệm.

Vai trò & Đóng góp: Nhà phát triển độc lập (100%). Tự lên ý tưởng bố cục, thiết kế giao diện bằng CSS, lập trình cấu trúc HTML và tự cấu hình triển khai đưa website lên môi trường Internet qua hệ thống GitHub Pages.

⚙️ Công nghệ sử dụng: HTML5, CSS3 (Flexbox, Grid, Linear Gradient), Git, GitHub Pages[cite: 1]
